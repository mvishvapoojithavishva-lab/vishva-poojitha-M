1️⃣ Backend (Node.js + Express + MongoDB)
Step 1: Initialize Project
mkdir todo-app-backend
cd todo-app-backend
npm init -y
npm install express mongoose cors


2️⃣ Frontend (React + Hooks)
Step 1: Create React App
npx create-react-app todo-app-frontend
cd todo-app-frontend
npm install axios


3️⃣ Running the App

Start backend:
node server.js

Start frontend:
npm start
